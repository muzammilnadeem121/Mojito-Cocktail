# Lime Wave

**Lime Wave** is a visually rich landing page for a mojito & cocktails shop, built to feel as refreshing as the drinks it showcases.  
The site focuses on smooth motion, bold typography, and immersive scroll-based animations.

Built with **Vite + React**, animated using **GSAP**, and styled with **Tailwind CSS**, Lime Wave delivers a modern, high-performance UI experience.
